## Data dictionary for order data
| Column Name   | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| order_id      | Unique identifier for each order.                                               |
| customer_id   | Unique identifier for each customer.                                            |
| order_date    | Date when the order was placed.                                                   |
| category      | The category of the product ordered (e.g., Electronics, Clothing, Beauty). |
| sku           | Stock Keeping Unit, a unique identifier for each product.                          |
| quantity      | The number of units ordered.                                                   |
| unit_price    | The price per unit of the product.                                                   |
| total_amount  | The total amount for the order (quantity * unit_price).                                  |
| status        | The current status of the order (e.g., Delivered, Cancelled, Processing, Shipped).                                                   |
